# Döviz Çeviri – Python GUI

Python ve Tkinter kullanılarak geliştirilmiş basit ve kullanıcı dostu bir döviz çevirici.  
Geliştirici: **Kemal Karatoprak**

## Özellikler
- TL ↔ USD, EUR, GBP dönüşümü  
- İki yönlü döviz çevirimi (TL’den dövize ve dövizden TL’ye)  
- Güncel döviz kurları API üzerinden alınır  
- Tkinter arayüzü ve özel ikon  
- “Hakkında” butonu ile geliştirici bilgisi  

## Python ile Çalıştırma
1. Python yüklü olduğundan emin ol (>=3.8 önerilir)  
2. Gerekli kütüphaneyi yükle:  
   ```bash
   pip install requests
3. Programı çalıştır:

python main.py
