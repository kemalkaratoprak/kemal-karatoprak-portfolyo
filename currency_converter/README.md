# Döviz Çeviri – Python GUI

A simple and user-friendly currency converter developed by **Kemal Karatoprak** using Python and Tkinter.

## Features
- TL ↔ USD, EUR, GBP conversion
- Bidirectional currency conversion
- Real-time exchange rates via API
- Tkinter GUI with custom icon
- “Hakkında” button with developer info
